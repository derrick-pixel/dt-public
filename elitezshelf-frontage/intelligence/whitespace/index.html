Artefact 01 · Strategy canvas

8 dimensions × 5 actors · 0–5 scoring

Customer evaluation axes — not feature checklists. The cluster reveals where incumbents converge. Our line breaks the cluster on three axes, deliberately concedes one (image-AI maturity).

Scoring rubric

  • Audit cadence

    0 = ad-hoc · 3 = monthly panel · 5 = weekly passive on routed visits

  • Retailer coverage (SG)

    0 = none · 3 = limited fixed-cam footprint · 5 = ≥60% organised retail weekly

  • Manpower model

    0 = none · 3 = contracted auditors · 5 = existing routed merchandisers, zero marginal

  • Time to first dashboard

    0 = >1yr · 3 = 8wk · 5 = 4wk

  • Asset-light (low CapEx)

    0 = $300K+ CapEx · 3 = mid · 5 = body-cam only (~S$280)

  • Image-AI maturity

    0 = none · 3 = mid · 5 = >13yr training data, broad SKU lib

  • SG localisation

    0 = global only · 3 = SG entity · 5 = SG-HQ, SG retailer relationships, PDPA-registered

  • Planogram diff depth

    0 = none · 3 = facings only · 5 = facings + sequence + eye-level + adjacency

Caveats

Where we deliberately concede.

regional_panel = NielsenIQ

NielsenIQ Brandbank's purchase-side panels are not ours. We are a complement, not a replacement, for monthly panel data — and we will say so to the CFO.

attack #1 only · MNCs

The top-3 MNCs (Coca-Cola, Unilever, Nestlé) already run multi-year Trax contracts. We do not lead with them; we land via mid-market, niche imports, and DTC graduates.
