competitor-intel-template

Reusable static-site template for competitor / market analytics. Eight named Claude Code subagents. Downloadable full-bleed PDF report.

Live demo GitHub

Get started in 5 steps

Fork the template, brief Claude, run the agents, ship the PDF, curate the template forward.

  1. 1

    Fork

    Use GitHub's Use this template button or gh repo create --template. Enable Pages on main / root.

  2. 2

    Brief Claude

    Open claude in the new repo. Paste your company, market, and competitor scope. Claude reads AGENT.md.

  3. 3

    Run agents 1–6

    Research → market → pricing → whitespace → design audit → viz. Each agent writes to its slice of template/data/ and commits.

  4. 4

    Ship the report

    Dispatch report-generator (agent 8). It compiles agents 1–6 into the full-bleed downloadable PDF.

  5. 5

    Curate back

    Run methodology-curator (agent 7). It diffs the finished project against the template and proposes improvements to merge upstream.
