# Elitez Site Supervisor

A regulated talent-supply vertical extension of Elitez Group — supplying BCA-licensed Resident Engineers (REs), Resident Technical Officers (RTOs), and Qualified Persons (Supervision) to Singapore construction sites.

**Positioning:** Tier-3 challenger at the boundary of Tier-2. Operator-led, license-track, training-anchored.

**Strategic anchor:** The 3 September 2025 BCA announcement made firm-based site supervision licensing mandatory from 2028 for projects above S$75M. We are filing for SAC accreditation in 2026 and BCA licence by Q1–Q2 2028.

## Site

- **Public:** company profile, four service tiers (Supply / Manage / Certify / Train), compliance & licence-track timeline, Adept Academy WSQ Built Environment track, sectors, operator pedigree.
- **Admin:** competitor intelligence, market sizing funnel, pricing & unit economics, whitespace blue-ocean atlas, design audit, board-grade report. Built on the [`competitor-intel-template`](https://github.com/derrick-pixel/competitor-intel-template) 9-agent methodology.

## Stack

Vanilla HTML5 / CSS3 / ES2020 modules. No build step. Chart.js + html2canvas + jsPDF via CDN/vendored. GitHub Pages deploy.
