# Favicon — Past Use

## Passage
- **Design:** Simple dove silhouette on warm-cream background.
- **Strategy:** Icon mark + solid accent + complete format set.
- **Why it works:** The dove reads at 16×16 because the silhouette is distinctive; warm cream stands out against both light and dark browser tabs.

## Altru
- **Design:** 🧧 red packet emoji.
- **Strategy:** Emoji + inverted + standard format.
- **Why it works:** Emoji favicons are a legitimate shortcut for consumer / playful brands. Cross-platform consistent, zero design effort, instantly recognizable.

## Vector Sky Defense
- **Design:** Targeting bracket mark (the same shape used in the hero canvas).
- **Strategy:** Icon mark + solid amber on black + adaptive SVG.
- **Why it works:** The bracket is 4 right-angle shapes — dead simple at 16×16. Amber reads on both tab backgrounds.

## What to avoid (seen in past projects)

- Company-name wordmark shrunk to 16×16 → mush. Use initial-letter instead.
- Complex 3D render as favicon → renders as a pixel blob. Flatten to 2D mark first.
- Transparent PNG only → works in Chrome but Firefox's tab background shows through in dark mode, making fine detail invisible. Always include a solid-bg fallback.
