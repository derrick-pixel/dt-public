# Game — Mechanic Fit

| Mechanic | Fit | Notes |
|---|---|---|
| og-social-meta | core | Mandatory |
| canvas-hero | core | Most games want a visual hero / game canvas |
| localstorage-state | core | Save game |
| wizard-form | core | Onboarding / character creation |
| admin-auth-gate | optional | For game master / dev tools |
| multi-page-scaffold | optional | For non-game sections (about, leaderboard) |
| chartjs-dashboard | optional | Stats screens |
| design-variant-switcher | optional | For pre-launch aesthetic exploration |
| pdf-pipeline | rare | Certificates on completion |
| formspree-form | rare | Only for post-game feedback capture |
| paynow-qr | rare | Monetization only |
| admin-insights-panel | rare | Games rarely need pricing strategy docs |
| dashboard-admin-shell | rare | For live-ops GM tools only |
| streamlit-analytics-scaffold | rare | Not the right toolchain for games |
